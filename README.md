# CopyShare

CopyShare 是一款面向局域网的剪贴板同步和文件传输工具。

两台电脑在同一局域网内连接并互相信任后，可以同步剪贴板内容，也可以在电脑之间传输文件，适合多台设备办公的人。

> 不需要公网服务器，不走云端中转，适合在同一个 Wi-Fi、路由器或办公局域网内使用。

## 主要功能

- **局域网自动发现设备**：自动扫描同网段 CopyShare 设备，发现后显示为“待连接”。
- **设备信任机制**：只有双方确认信任后，才允许同步剪贴板和传输文件。
- **剪贴板同步**：支持文本、截图和图片剪贴板同步。
- **PC-PC 文件传输**：已信任电脑之间可直接传输文件。
- **手机临时连接**：电脑生成二维码，手机扫码后可临时取用或提交剪贴板内容。
- **浮窗模式**：可切换为小浮窗，快速查看同步状态和最近剪贴板内容。
- **桌面通知**：支持剪贴板同步、设备上线/离线、文件传输结果等提醒。
- **开机自启**：可设置开机自动启动和启动后自动同步。

## 快速开始

### 1. 打开 CopyShare

在两台电脑上都启动 CopyShare，并确保两台电脑连接到同一个局域网。

### 2. 查找局域网设备

进入 **设备连接** 页面，点击 **扫描局域网设备**。

发现到的设备会显示为 **待连接**。

如果没有自动发现，也可以手动输入对方电脑的局域网 IP 和端口连接。

### 3. 连接并信任设备

点击目标设备的 **连接**。

连接成功后，双方设备都需要确认 **信任设备**。

只有互相信任后，剪贴板同步和文件传输才会生效。

### 4. 开始同步剪贴板

回到 **总览** 页面，确认状态为 **同步中**。

之后在任意一台电脑复制文本、截图或图片，另一台电脑会自动收到。

### 5. 传输文件

进入 **文件传输** 页面：

1. 选择要发送的文件。
2. 选择目标设备。
3. 点击发送。
4. 对方接收后，文件会保存到本机下载目录下的 `Copy-Sharer` 文件夹。

## 手机连接

进入 **手机连接** 页面，点击 **生成二维码**。

手机扫码后会打开一个局域网页，可临时查看电脑剪贴板内容，也可以把手机内容提交到电脑剪贴板。

手机连接只在本次运行期有效，关闭会话或退出程序后失效。

## 浮窗模式

点击顶部的 **切换浮窗** 可进入小浮窗模式。

浮窗可显示：

- 当前同步状态
- 已连接设备数量
- 网络状态
- 最近剪贴板内容

点击浮窗里的 **主面板** 可回到完整窗口。

## 常见问题

### 扫不到局域网设备怎么办？

请检查：

- 两台电脑是否在同一个 Wi-Fi 或局域网。
- 对方是否已经打开 CopyShare。
- Windows 防火墙是否允许 CopyShare 访问专用网络。
- 是否连接到了 VPN、虚拟网卡或隔离网络。
- 可尝试手动输入对方 IP 和端口连接。

### 连接成功但不能同步怎么办？

请检查：

- 双方是否都已点击 **信任设备**。
- 总览页状态是否为 **同步中**。
- 设置中是否开启了对应的同步内容。
- 对方设备是否仍在线。

### 文件传输失败怎么办？

请检查：

- 双方设备是否已互相信任。
- 文件是否超过限制。
- 下载目录是否有写入权限。
- Windows 防火墙是否阻止局域网连接。

### 手机扫码打不开怎么办？

请检查：

- 手机和电脑是否在同一个局域网。
- 手机浏览器是否能访问电脑局域网 IP。
- 电脑防火墙是否允许 CopyShare 的局域网访问。

## 安全说明

- CopyShare 的设备发现、剪贴板同步和文件传输都面向局域网。
- 文件传输不使用公网服务器，不上传到云端。
- 未信任设备不能同步剪贴板，也不能传输文件。
- 建议只在可信网络中使用，不要信任陌生设备。
- 剪贴板可能包含密码、验证码或隐私内容，使用前请确认对方设备可信。
